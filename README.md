ENG
A program that separates CurseForge modpacks into 3 groups:
1)Mods
2)Shaders
3)Resource Packs
Why?
Not everyone knows how to use modpacks, and not every launcher works with them correctly.

You can get your API key for free on the website:
https://console.curseforge.com/#/api-keys
Just log in or create a CurseForge account.
The program initially uses one public key with a limit of 5 parallel downloads.
Each key added by the user increases the limit by 10 downloads.
Your API keys are stored locally in the config.json file and are not sent anywhere.

RU
Программа, которая разлагает модпак из CurseForge на 3 группы:
1)Моды
2)Шейдеры
3)Ресурспаки
Зачем?
Не все умеют пользоваться модпаками, и не каждый лаунчер корректно с ними работает.

Получить свой API-ключ можно бесплатно на сайте:
https://console.curseforge.com/#/api-keys
Достаточно войти или создать аккаунт CurseForge.
Изначально в программе используется 1 публичный ключ с лимитом 5 параллельных загрузок.
Каждый добавленный пользователем ключ увеличивает лимит на 10 загрузок.
Ваши API-ключи сохраняются локально в файле config.json и никуда не отправляются.
